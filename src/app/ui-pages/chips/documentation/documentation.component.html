Chips

ntx-chip is a component for building lists of strings or objects. The list items are displayed as 'chips'.

Chips types

There are four types of nxt-chips: use ntx-image-chip if you need chip with images, use ntx-category-chip if you need to specify the category of the content, use ntx-basic-chip for basic chips, and use ntx-filter-chip for chips that are used like radiobutton an only one of them can be selected.

Configuration

Color and disabled attributes can be configured in each type of chip, just by adding the atribute [color] or [disabled] in the html tag. In .ts file we have added these variables as strings so they can be configured.

In .ts file of each component, we have added an array to repeat x times the items and have a better perception of it's final look&feel. Feel free of configure this file as you wish.
