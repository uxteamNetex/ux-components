Lists

ntx-list is a component that wraps and formats a series of line items.

Lists types

There are several types of nxt-list: use ntx-list-one-line-avatar for lists that need just one image and one line of content. Use ntx-list-two-lines-avatar for lists that need have an image and two lines of content: title and subtitle. Use ntx-list-two-multilines-icon for lists that represent sections, where the elements to show are its image, title and an indeterminate number of links to the different subsections. Use ntx-list-one-line-options for lists that have a loading process and options. Use ntx-list-one-line-navigable for lists that are clicable and navigable. You can use ntx-list-horizontal for lists that have title and subtitle displayed in a horizontal way. And use ntx-list-sidemenu for lists that allows us to navigate between different views of a side menu.

Configuration

The content of each list is being shown in its own .ts file, feel free to move this content to a external json file.
