Release v4.0.1: Error pages, progress spinner and more

29 april 2019

What’s new section is our monthly update that highlights recent product improvements we’ve made so you can easily stay up to date on what’s new.

What’s new

Empty pages

We start the new version of the library with the new ntx-empty-page component to put on our empty screens and tell the user that there is still no content on that screen.

Error pages

Our next novelty are the needed error pages. These are the components:

  • Error page 403 — Error pages 403 will be represented with the component ntx-error403
  • Error page 404 — Error pages 404 will be represented with the component ntx-error404
  • Error page 500 — Error pages 500 will be represented with the component ntx-error500
  • Error page 503 — Error pages 503 will be represented with the component ntx-error503

New icons

With this update we have a new batch of icons. For this, several css files have been touched, which you must update to see the icons correctly:

  • ux-components/src/styles/app/utils.scss
  • ux-components/src/styles/app/variables.scss
  • ux-components/src/styles/fonts/styles.scss
  • ux-components/src/styles/fonts/variables.scss
  • ux-components/src/assets/fonts/icons/netex.eot
  • ux-components/src/assets/fonts/icons/netex.svg
  • ux-components/src/assets/fonts/icons/netex.ttf
  • ux-components/src/assets/fonts/icons/netex.woff

Chips

We released a new chip model, ntx-image-subtitle-chip that contains image, title and subtitle to provide the most relevant information at a glance.

Progress spinner

We already have in the library the ntx-progress-spinner component to tell our users that there is a loading process in progress.

Card 14

We launched a new version of cards, ntx-card14 component that is a version of the card8 that indicates that the card has already been added.
