Progress bar

ntx-progress-bar is a horizontal progress-bar for indicating progress and activity.

Configuration

Color attribute can be configured in each progress bar, just by adding the atribute [color] in the html tag. In .ts file we have added this variable as strings so it can be configured.

The primary color can be used by specifying it in color atribute [color]="'primary'". For secondary colors, you should use the accent value [color]="'accent'".

The value of the progress bar is shown by an atribute in the html. Feel free to change that in order to program its current value.
