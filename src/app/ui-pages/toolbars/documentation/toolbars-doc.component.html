Toolbars

ntx-toolbar is a component used to place a toolbar in your app.

Toolbars are usually used above a content area to display the title of the current page, and show relevant action buttons for that page.

Toolbars types

There are various types of nxt-toolbar:

  • ntx-toolbar is a basic toolbar.

  • ntx-toolbar-searchboxis a toolbar with a searchbox with filter panel.

  • ntx-toolbar-backis a toolbar with a return button.

  • ntx-toolbar-tabsis a toolbar with a second level of navigation.

  • ntx-toolbar-back-tabsis a toolbar with a second level of navigation, a return button and conditionally an action button.

Configuration

Keep in mind that each of these components need more components to be complete. This is the case of the options menu of the user profile or the tabs. In ntx-toolbar.module.ts file all modules needed are imported.

Customization

Each Netex application has it's corporate color that must be seen in the toolbar. If you need the application to have the Talent Cloud color, you have to put the theme-talent class in the body, and if you need the application to have the Content Cloud color, you have to put the theme-content class in the body.
