Buttons

ntx-button is a component for building buttons and links.

Buttons types

There are several types of nxt-button: ntx-strocked-button, ntx-strocked-icon-button, ntx-strocked-toggle-button, ntx-text-button, ntx-text-icon-button, ntx-raised-button, ntx-raised-icon-button, ntx-icon-button, ntx-fab-button, ntx-mini-fab-button, ntx-square-button y ntx-fab-speed-dial-button.

Configuration

Color and disabled attributes can be configured in each type of button, just by adding the atribute [color] or [disabled] in the html tag. In .ts file we have added these variables as strings so they can be configured.

The primary color can be used by specifying it in color atribute [color]="'primary'". For secondary colors, you should use the accent value [color]="'accent'".

In .ts file of each component, we have added an array to repeat x times the items and have a better perception of it's final look&feel. Feel free of configure this file as you wish.
