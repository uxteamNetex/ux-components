Release v4.2.0: New cards, lists, paginator, toolbars and more...

05 may 2020

What’s new section is our monthly update that highlights recent product improvements we’ve made so you can easily stay up to date on what’s new.

What’s new

New card component

We released a new card componentized component, ntx-card18.

New list component

We released a new list componentized component, ntx-one-line-list.

New progress bar component

We released a new progress bar componentized component, ntx-progress-bar-level.



New list component

We released a new list componentized component, ntx-list1.

New input component without label

We released a new input component, ntx-input-no-label without label.

Paginator

We released a paginator component, ntx-paginator used with tables and/or grid of items like cards.

Search box with filter panel

We released a new of search box which includes a filter panel, ntx-searchbox-filter-panel

Link button component

We released a new type of button, ntx-link-button that allows us to navigate to some route or perform some action and can show an icon conditionally.

action!

New icons

With this update we have a new batch of icons. For this, several css files have been touched, which you must update to see the icons correctly:

  • ux-components/src/styles/app/utils.scss
  • ux-components/src/styles/app/variables.scss
  • ux-components/src/styles/fonts/styles.scss
  • ux-components/src/styles/fonts/variables.scss
  • ux-components/src/assets/fonts/icons/netex.eot
  • ux-components/src/assets/fonts/icons/netex.svg
  • ux-components/src/assets/fonts/icons/netex.ttf
  • ux-components/src/assets/fonts/icons/netex.woff

Icon stroked buttons

We released a new type of button, ntx-stroked-icon-button that allows us to navigate between previous or subsequent elements.

List sidemenu

We released a new type of list, ntx-list-sidemenu that allows us to navigate between different views of a side menu.

Toolbar (added new types)

We released new type of toolbars, ntx-toolbar-back-tabs that includes a back button, a second level of navigation and conditionally an action button and ntx-toolbar-searchboxthat includes a searchbox with filter panel.
